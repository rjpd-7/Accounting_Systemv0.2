Journal Entries Basic Information

A Journal Entry records a financial transaction, made up of one or more lines (debits and credits) that must balance (total debits = total credits). The account names are from the Chart of Accounts.

First row should be a debit input.

Last row should be a credit input.

There shouldn't be empty debit and credit values

Action column has a remove button whenever you need to delete the row. Note that the 1st row cannot be removed.

You cannot simultaneously enter debit and credit values on the same row.
